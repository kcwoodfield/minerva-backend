# Minerva Backend

## Get Started

Create virtual environment

```
python3 -m venv venv
```

Activate the virtual environment

```
source venv/bin/activate
```

Install Django

```
pip install django django-ninja
```

Run server

```
cd src && python manage.py runserver 8001
```


## Project Resources

- [Django Ninja](https://django-ninja.dev/)
